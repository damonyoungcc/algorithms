## QuickFind

### 思路

1. 初始化一个长度为`N`的数组 id，初始值 `id[i] = i`
2. `FIND`: 传入 `p` 和 `q` 判断是否有相同的`id`
3. `Union`: `union(p, q)`

- 初始将 `id[p]`设置的值为`q`
- 推广一下，每次都遍历数组，将`id[i] === pid`的项，都把值 `id[i] = qid`
