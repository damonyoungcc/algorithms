## QuickUnion

### 思路

1. 初始化一个长度为`N`的数组 id，初始值 `id[i] = i`
2. `FIND`: 传入 `p` 和 `q` 判断是否有相同的`id`
3. `Union`: `union(p, q)`

- 初始化 `id[p]` 的父级 为 `id[q]`
- 每次都溯源 `id[p]` 的 `id[proot] = id[qroot]`
- 把 `p` 的 `proot id[proot] = qroot`
- 查找时，查找 `p q` 是否有相同的 `root`
